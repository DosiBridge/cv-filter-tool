# CV Filter Tool - Frontend

Next.js frontend application for CV filtering and matching.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env.local` file (optional):
```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Running

```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Build

```bash
npm run build
npm start
```

## Features

- Upload multiple CV files (PDF, DOC, DOCX)
- Enter job requirements
- AI-powered CV matching
- Results sorting and reordering
- Detailed match analysis with strengths and weaknesses
